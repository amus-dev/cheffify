_Cheffify es un proyecto de ecommerce diseñado para entregar platos preparados y de sabor casero directamente a domicilio. Este sitio web está creado con la tecnología moderna y eficiente de Astro, combinada con Tailwind CSS para un diseño responsive y atractivo. La plataforma ofrece una experiencia de usuario intuitiva y fácil de navegar, permitiendo a los clientes disfrutar de una amplia variedad de comidas familiares con solo unos clics._
